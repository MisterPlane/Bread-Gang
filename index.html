About the server

This server started as a private server, with very little to offer. All this started because of few jokes about bread, then we decided to make a plugin about it, and now we have grown to a public server with few but very loyal players. We are proud of what we reached so far, and we hope this growth never stops. Thank you, everyone who stayed with us from the beginning.
